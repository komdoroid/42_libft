## Description
Libft is a custom C library.
The goal of this project is to understand how C standard functions work by learning how to reimplement them myself and learning how to use them effectively.

## Instructions

### Compilation
To compile the library:
```bash
make
```
This will generate `libft.a`

### Available Commands
```bash
make        # Compile the library
make clean  # Remove object files
make fclean # Remove object files and library
make re     # Recompile everything
```
